## English

### About this repository

This repository is a curated, community-friendly collection of command-line (CLI) tools for developers, system administrators, and technology enthusiasts. It started as a base list of 50 essential tools organized by category (media, networking, data processing, system monitoring, productivity, and development), and has since been **expanded** with additional tools that cover the same problem spaces from new angles — from established market leaders in specific niches to promising open-source and AI-powered alternatives.

### Purpose

The goal is to help people discover the right tool for the job quickly, without wading through endless "awesome list" repetition. Each expansion entry follows a consistent format — category, one-line differentiator, key strength, and target audience — so you can scan and compare options at a glance instead of googling every tool individually.

### Lists

- 📄 [Expanded list — English](./cli-tools-expansion-en.md)
- 📄 [Lista expandida — Português (Brasil)](./ferramentas-cli-expansao-pt-br.md)

### Contributing

Suggestions for new tools are welcome. When proposing an addition, please follow the existing format (category, description, strength, target audience) and avoid duplicating tools already listed.

## Português (Brasil)

### Sobre este repositório

Este repositório é uma coleção curada e amigável de ferramentas de linha de comando (CLI) para desenvolvedores, administradores de sistema e entusiastas de tecnologia. Ele começou como uma lista-base com 50 ferramentas indispensáveis organizadas por categoria (mídia, redes, processamento de dados, monitoramento de sistema, produtividade e desenvolvimento) e, desde então, foi **expandido** com novas ferramentas que resolvem os mesmos problemas sob novos ângulos — desde líderes de mercado consolidados em nichos específicos até alternativas open-source e baseadas em IA promissoras.

### Propósito

O objetivo é ajudar as pessoas a encontrar rapidamente a ferramenta certa para cada tarefa, sem precisar navegar por listas repetitivas do tipo "awesome list". Cada item da expansão segue um formato consistente — categoria, diferencial em uma frase, ponto forte e público-alvo — para que você possa comparar as opções de forma rápida, sem precisar pesquisar cada ferramenta individualmente.
